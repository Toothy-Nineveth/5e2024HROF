# Heroes of Faerûn RPG Companion App
This project contains structured game content from the **Dungeons & Dragons 2024** ruleset, specifically focused on material from the *Forgotten Realms: Heroes of Faerûn* release.

## Contents

This repository includes organized data files for:

* **Subclasses**
* **Feats**
* **Spells**
* **Backgrounds**

All material is adapted for use within the companion app to help streamline gameplay and character creation.
